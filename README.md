# Shape-UP Nutrition Centre — Landing Page

Tamil-language landing page for Shape-UP Nutrition Centre, a nutrition & lifestyle coaching business in Gerugambakkam, Chennai, run by Nalini R.

## Structure
- `index.html` — single-page site (Tamil)
- `styles.css` — design system (colors, type, layout)
- `script.js` — lead form → WhatsApp handoff
- `assets/` — logo mark + ribbon divider (SVG)

## Stack
Static HTML/CSS/JS — no build step. Deployed on Vercel.
